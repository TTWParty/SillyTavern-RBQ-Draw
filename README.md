# SillyTavern RBQ生图扩展 (RBQ-Draw)

一款专为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 打造的高级沉浸式图像生成插件。深度定制并完美集成了 **NovelAI V4.5** 与 **ComfyUI** 的全部高级特性，在聊天界面内为你带来官方原生级别的无缝动态生图体验。

---

## ✨ 核心特性

### 🎨 终极 NovelAI V4.5 体验
- **完整高级参数**：无缝支持 V4.5 特有的 `Variety+`、`Decrisper`、`SMEA` 以及 `SMEA DYN` 等高级采样限制控制。
- **Vibe Transfer (氛围转移) 完美复刻**：
  - 支持最高 6 张 Vibe 参考图或 1 张 Precise Reference (精确参考) 提取。
  - **自动特征提取**：上传原图点击生成时，插件会自动在后台无感完成 `/encode-vibe` 特征提取。
  - **完全免算力复用**：提取过的参考图会自动缓存为 `.naiv4vibe` 文件。支持**一键导出与二次导入**，直接使用本地 `.naiv4vibe` 文件进行后续生图**实现 0 附加 Anlas 算力消耗**！
- **原生级沉浸交互**：抛弃简陋的全屏遮罩，采用与 NovelAI 官方网页版 1:1 像素级还原的行内动态加载光圈，并精准提供 `正在请求集群分配运算资源...`、`正在接收服务器下发的数据...` 等全阶段状态播报。

### ⚙️ 极客级 ComfyUI 调度
- **全动态工作流支持**：不仅支持官方格式流，更支持自定义节点映射。你可以在插件面板里直接修改你的 JSON 工作流字典。
- **自动模型抓取**：自动从 ComfyUI 后端拉取可用的 Checkpoints 列表并在面板中即时切换，无需繁琐复制。

### 💬 无感聊天流整合
- **行内触发器**：支持正文解析生图，小说/扮演中通过 `[scene]一段场景描述[/scene]` 标签，AI 助手发送后插件自动捕获并后台无感出图。
- **自由悬浮球**：界面边缘的悬浮法杖，支持自由拖拽。随时点击**"快速生成"**或者一键预览当前构建的完整 Prompt，不错过任何灵感。
- **历史画廊系统**：点击聊天流内的缩略图即可唤出沉浸式画廊，自带下载、放大、翻页及缓存容灾机制。

## 📦 安装说明

本插件已做极致体积优化，支持直接在 SillyTavern 内一键通过 URL 安装：

1. 打开 SillyTavern 的 **扩展 (Extensions)** 面板。
2. 点击顶部的 **安装扩展 (Install Extension)**。
3. 在弹出的输入框内填入本 GitHub 仓库地址：
   ```text
   https://github.com/TTWParty/SillyTavern-RBQ-Draw
   ```
4. 点击 **保存并安装**。
5. 重启 SillyTavern 服务器即可生效。

## 🛠️ 配置与使用方法

安装并重启完毕后，点击 SillyTavern 顶部导航栏的 **扩展面板 (积木图标)**，找到 **RBQ生图** 展开其设置，或直接点击界面上出现的悬浮圆球进入**插件设置**。

### 1. 基础生图设置 (NAI 模式)
1. 在配置面板中将 **当前模式** 切换为 `NovelAI (包含中转)`。
2. 填写 **接口地址**（如：`https://api.novelai.net/ai/generate-image`，如果你使用中转 API，请填入对应的中转端点）。
3. 填写你的 **API Key (鉴权密钥)**。
4. (可选) 设定全局的前缀/后缀/反向提示词。

### 2. 聊天流自动出图
你可以将插件配置为**自动模式**或**手动模式**：
- **匹配规则**：插件默认会监听含有 `[scene]...[/scene]` 或是 `[img]...[/img]` 的文本块。
- 你可以修改这部分匹配正则，支持完全自定义触发词。

### 3. 使用 Vibe Transfer
- 在生成的历史画廊或配置面板中找到 **Vibe** 功能区。
- 点击 **上传图像 / 导入 Vibe 特征**。
- 设置每张 Vibe 图的信息权重 (Information Extract) 和参考强度 (Reference Strength)。
- 点击生成，插件会自动处理打包。生成满意的结果后，记得点击历史记录里的 **"导出"** 按钮将 `.naiv4vibe` 文件保存到本地，以便下次白嫖特征算力！

## ❓ 常见问题

**Q: 点击生成后，很久没出图，或者一直显示加载中怎么办？**  
A: 请按 `F12` 打开浏览器开发者工具查看 Console 报错，同时检查你的后端 API 地址是否填对，是否需要科学上网，以及 API Key/Token 的余额是否充足。

**Q: 刷新浏览器后，之前生成的 NAI 图片变成了问号？**  
A: 这是早期版本的浏览器 Blob 缓存丢失导致。请确保你安装的是最新版本，插件已内置 IndexedDB 容灾机制，会自动从数据库中恢复历史丢失的图片。
