# RBQ生图扩展 (SillyTavern-RBQ-Draw)

一款专为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 打造的全能级高级图像生成插件。支持 **NovelAI、ComfyUI** 以及 **第三方兼容生图接口**。它在酒馆原有的聊天界面中深度植入了令人惊叹的原生级动态 UI，并提供了从“全自动无感生图”到“高阶节点级控制”的完整工作流程。

---

## 🚀 平台支持与核心能力

本插件采用了模块化的独立参数隔断设计。无论你在哪种模式下切换，它们的分辨率、模型、控制参数都会被**严格隔离保存，互不干扰**。

### 1. 🎨 NovelAI 终极套件体验
专为 NAI V4.5 深度定制，带来比官方网页版更激进的使用体验：
- **完整 V4.5 参数引擎**：无缝支持 `Variety+`、`Decrisper` (去噪点)、`SMEA` 以及 `SMEA DYN` 特殊限制开关。
- **神级 Vibe Transfer (氛围转移)**：
  - 完美复刻官方的多图 Vibe / 单图精确参考 (Precise Reference) 功能。
  - **自动隐式提纯**：上传原图点生成，插件在后台自动调用 `/encode-vibe` 提取特征。
  - **免算力白嫖机制**：提纯过的特征会自动缓存为 `.naiv4vibe` 文件。你可以在画廊中随时**"一键导出"**。下次只要上传这个特征文件，就能**以 0 Anlas 附加算力的代价**实现 Vibe 转移！
- **真·无缝压缩包解析**：内置纯前端 JSZip 解析库。突破任何 API 中转或网关拦截造成的二进制流损坏，彻底告别图片发不出来、图裂和黑屏。

### 2. 🔌 ComfyUI 极客调度模式
将你的本地/云端 ComfyUI 工作流直接无缝嵌入到酒馆聊天中：
- **自动获取模型**：填写你的 ComfyUI 地址后，插件会自动通过 API 抓取后端的 Checkpoint 列表，并在酒馆下拉菜单中实时显示，支持点击一键切换模型。
- **全动态工作流绑定**：在设置面板中，你可以直接粘贴官方或第三方的 JSON 工作流字典。
- **智能变量映射**：内置了 `{{prompt}}`, `{{negative_prompt}}`, `{{model}}`, `{{width}}`, `{{height}}`, `{{seed}}` 等宏变量。无论你的连线多么复杂，只需在对应的节点注入变量，插件就能在每次生成时自动根据聊天内容注入动态词条！

### 3. 🌐 通用接口模式 (Free Mode)
对于使用第三方或其它 SD 引擎转发站点的用户：
- 提供极简纯净的生成端点配置。
- 支持高度自定义的发包映射，兼容市面上大多数标准 /generate-image 规范的黑盒 API。

## 🌟 极致的沉浸式交互 (UI/UX)

本插件彻底抛弃了简易出图插件“一张全屏黑幕遮挡视线”的粗暴做法：
- **官方级加载光圈**：我们 1:1 像素级复刻了 NovelAI 官方的深色底边光圈加载动画。
- **全阶段状态播报**：在按钮下方精准显示当前执行的后端操作：`正在打包特征组...` -> `正在请求集群分配运算资源...` -> `正在接收服务器下下发的数据...`，一切尽在掌握。
- **全能图像画廊**：
  - 生成结果以优雅的行内缩略图卡展示在聊天流中。
  - 点击即可唤出沉浸式黑底画廊，支持 **滑鼠滚轮缩放、随意拖拽平移视图、左右切换历史版本**。
  - 画廊底部直充原生下载按钮。
- **IndexedDB 持久化缓存容灾**：所有出图结果会自动写入浏览器底层数据库。**即使直接 F5 刷新网页，你之前的生成结果也不会丢失、不会变成问号图片**，随时可以找回并下载原图。

## 🎯 触发方式机制

我们为你提供了三种符合不同扮演习惯的出图方式：

### 1. 文本流捕获（行内触发 / 自动出图）
默认监听助手或你的文本中包含的特定格式：
- `[img]一段画面描述[/img]`
- `[scene]人物坐在酒吧吧台前，端着酒杯[/scene]`

只要 AI 跑出了这个格式，插件会**自动拦截并隐秘生成**，无需手动点任何按钮。这让你在跑长篇 RP 或连载剧情时获得极其沉浸的“附带插画”体验。你可以完全在设置里自定义你想捕获的前缀和后缀正则。

### 2. 悬浮魔法球交互
屏幕右侧会有一个永远在线的动态悬浮球（支持自由拖拽）：
- 点击展开快捷菜单：
  - **一键预览提示词**：立刻查看当前组装好的（注入了各种系统前缀的）完整英文 Prompt，方便 debug 查错。
  - **快速盲盒生成**：弹出一个简易输入框，输几个词，立马在后台触发绘制。
  - **直达配置面板**：无需翻找系统菜单，一键打开插件设置。

### 3. 手动按钮触发
在消息的工具栏里内置了生成的专属点击按钮，随时对某一条回答触发重绘。

## 📦 安装与配置

本插件已做极致压缩，可以直接在 SillyTavern 内部通过仓库直链一键安装。

1. 打开 SillyTavern，点击顶部最右侧的 **扩展插件模块 (积木图标)**。
2. 点击上方的 **安装扩展 (Install Extension)** 按钮。
3. 在地址栏中直接粘贴本仓库地址：
   ```text
   https://github.com/TTWParty/SillyTavern-RBQ-Draw
   ```
4. 点击 **保存并安装**，安装成功后重启你的 SillyTavern 服务端/刷新页面即可生效。
5. 成功后，点开界面右侧的悬浮魔法球或扩展列表的齿轮图标，即可开始配置你的接口 URL！
