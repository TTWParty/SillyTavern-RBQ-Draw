# RBQ生图扩展 (SillyTavern-RBQ-Draw)

专为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 打造的全能图像生成扩展。支持 NovelAI、ComfyUI 及通用生图接口，为你带来原生级别的聊天互动生图体验。

---

## 📦 快速安装

1. 打开 SillyTavern，点击顶部最右侧的 **扩展 (Extensions)** 按钮（积木图标）。
2. 点击 **安装扩展 (Install Extension)**。
3. 在地址栏中直接粘贴本仓库地址：
   ```text
   https://github.com/TTWParty/SillyTavern-RBQ-Draw
   ```
4. 点击 **保存并安装**，安装成功后重启酒馆即可生效。

---

## 📖 使用指南

### 1. 基础配置
- 点击右侧界面边缘的 **悬浮魔法球**，或从顶部扩展列表进入 **RBQ生图** 配置面板。
- 选择你需要使用的接口模式（NovelAI、ComfyUI 或 默认）。
- 填入你自己的 API 端点地址和对应的 Token 密钥。

### 2. 开始生图 (三种方式)
- **对话流全自动出图**：配置好触发词后，只要模型或你在聊天中发送了类似 `[scene]人物描述[/scene]` 或 `[img]人物描述[/img]` 格式的文本，扩展会自动触发并在消息下方生成对应的图片。
- **悬浮球快速生成**：随时点击屏幕边缘的悬浮球菜单，选择“快速生成”，输入描述词即可立刻出图。
- **消息手动重绘**：在任意一条聊天记录的操作栏里，点击专属生成按钮即可触发重绘。

### 3. Vibe Transfer (氛围转移 - 仅 NAI 模式)
- 在画廊记录或配置中打开 Vibe 面板。
- 支持上传多张 Vibe 参考图或单张精准参考图 (Precise Reference)。
- 可以调整信息提取和参考强度，生成后点击“**导出**”即可保存提取好的 Vibe 特征文件，供下次一键导入使用。

## ✨ 主要功能介绍

- **NovelAI 全面支持**
  完美支持 NAI 最新版本特性，提供独立隔断的全局参数调整（`Variety+`、去噪、特殊限制等），互不干扰。
  
- **ComfyUI 极客定制**
  可以在面板中一键拉取你的云端大模型列表，支持直接填入动态的工作流。不论工作流多复杂，系统都会自动为你填入提示词。
  
- **原生级沉浸交互**
  - **加载动画**：与官方网页版一致的行内沉浸式深色加载光圈，并带有一秒不差的各个阶段状态流提示。
  - **历史画廊系统**：图片生成后会在消息历史中保留，随时点击可全屏查看、放大平移、左右切换画廊，支持一键下载原图。
  - **本地记录防丢**：生成过的图片全程受到本地缓存保护，断网或刷新浏览器依然可以找回原图。
