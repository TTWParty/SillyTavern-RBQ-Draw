# RBQ生图扩展 (SillyTavern-RBQ-Draw)

一款专为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 打造的全能级高级图像生成插件。在聊天界面内为你带来原生级别的无缝动态生图体验，支持 **NovelAI、ComfyUI** 以及 **第三方兼容生图接口**。

---

## 📦 安装说明

本插件支持直接在 SillyTavern 内一键通过 URL 安装：

1. 打开 SillyTavern，点击顶部最右侧的 **扩展插件模块 (积木图标)**。
2. 点击上方的 **安装扩展 (Install Extension)** 按钮。
3. 在地址栏中直接粘贴本仓库地址：
   ```text
   https://github.com/TTWParty/SillyTavern-RBQ-Draw
   ```
4. 点击 **保存并安装**，安装成功后重启你的 SillyTavern 服务器即可生效。

---

## 🛠️ 配置与使用方法

安装并重启完毕后，在此处进行快速配置：点击 SillyTavern 顶部导航栏的 **扩展面板 (积木图标)**，找到 **RBQ生图** 展开其设置，或直接点击界面边缘出现的悬浮圆球进入**插件设置**。

### 1. 基础连接设置
1. 在配置面板中，选择你想用的**当前模式**（NovelAI、ComfyUI 或 Free通用模式）。
2. 填写 **接口地址**（如：`https://api.novelai.net/ai/generate-image`）。
3. 填写你的 **API Key (鉴权密钥)**。
4. 在各模式专属标签页下，可以随时调整出图分辨率和采样参数。模型切换支持自动读取后端列表，可一键下拉选择。

### 2. 三种出图方式
- **自动捕获出图**：当 AI 助手（或你的设定文本）在聊天中输出了如 `[scene]人物坐在吧台前[/scene]` 的格式词时，插件会自动拦截并直接后台出图，实现真正的图文并茂跑团体验。
- **悬浮魔法球快抽**：点击屏幕边缘的悬浮法杖，选择“快速生成”输入短词即可盲盒出图。
- **手动重新生成**：在任意聊天消息的底部工具栏中，点击专属的出图按钮即可针对该条消息触发绘图。

### 3. Vibe Transfer（氛围转移）进阶玩法
- 在历史画廊或配置面板中找到 **Vibe** 功能区，上传你想要参考的图片（支持多图）。
- 设置参考强度，点击生成即可。
- 生成满意的结果后，记得点击历史记录里的 **"导出"** 按钮保存你的专属 Vibe 特征包。下次直接导入这个特征包即可零门槛复刻！

## 🌟 核心功能一览

### 1. 极致交互体验
我们彻底抛弃了常见的生图插件“黑屏遮挡”做法：
- **行内无感加载**：完美复刻官方的加载动画，在消息流内动态显示，不妨碍你继续聊天。
- **精准状态播报**：时刻提醒当前是在请求排队、生成中、还是在下载图片，告别无脑转圈盲等。
- **沉浸式图库画廊**：生成的图片会以缩略图内嵌，点击即可唤出超大画廊，支持 **滑鼠滚轮缩放、鼠标拖拽平移、左右翻页对比历史记录**。
- **断线保护机制**：所有生成的图片拥有防丢失保护，即使你在图片还没存完时直接 F5 刷新了浏览器，图片画廊依然能完美找回重载。

### 2. NovelAI 终极套件
专为 NAI V4.5 深度定制，体验比肩官方：
- **完整高级参数引擎**：无缝支持 `Variety+`、`Decrisper` (去噪点)、`SMEA` 以及 `SMEA DYN` 等进阶开关。
- **神级 Vibe Transfer**：完美支持多图特征提取 / 单图精确参考 (Precise Reference)。
- **黑图/碎图终结者**：独立的高级解压机制，无视绝大部份网络中转拦截，和偶尔发不出来图的情况彻底说拜拜。

### 3. ComfyUI 极客工作流
你的神级节点，现在可以直接在酒馆里无缝调用：
- **全动态工作流绑定**：你可以将完整的官方或第三方 JSON 工作流填入插件面板。
- **智能变量注入**：插件内置了大量魔法宏语（如 `{{prompt}}`, `{{negative_prompt}}`, `{{model}}`, `{{width}}`, `{{height}}`, `{{seed}}`）。无论你的节点线连得多么复杂，只需在对应的输入框填入宏，插件每次都会精准填充并带入到你的定制工作流中！
