# RBQ生图扩展 (SillyTavern-RBQ-Draw)

一款专为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 打造的高级图像生成扩展。在聊天对话中无缝嵌入动态生图体验，内置 **NovelAI (NAI)** 与 **ComfyUI** 两大模式，并支持通过插件注册任意第三方生图后端。

---

## 📦 安装

在 SillyTavern 中一键安装：

1. 点击顶栏 **扩展插件 (积木图标)** → **安装扩展**。
2. 粘贴仓库地址：
   ```text
   https://github.com/TTWParty/SillyTavern-RBQ-Draw
   ```
3. 点击 **保存** 并重启 SillyTavern 即可。

---

## 🚀 快速上手

安装后，点击聊天界面边缘出现的 **悬浮按钮** 打开插件主面板，或在扩展面板中找到 **RBQ生图** 展开设置。

### 1. 选择模式并连接

| 模式 | 说明 | 需要填写 |
|------|------|---------|
| **NAI** | NovelAI 远程生图 | NAI Token（支持官方直连 / RBQ 接口 / 自定义端点三种连接方式） |
| **ComfyUI** | 本地 / 远程 ComfyUI 工作流 | ComfyUI 地址（如 `http://127.0.0.1:8188`），密钥可选 |
| **插件模式** | 第三方插件注册的自定义后端 | 由插件定义（通过插件市场安装后自动出现在模式列表中） |

### 2. 触发生图

插件通过识别聊天消息中的 **prompt 标签** 来自动或手动触发生图。内置支持以下格式：

```
[scene]1girl, sitting at bar[/scene]
[img]a cozy tavern interior[/img]
```

你也可以在设置中自定义标签（如 `<draw>...</draw>`）或填写自定义正则表达式来扩展触发规则。

当 AI 在对话中输出包含上述标签的内容时，插件会：
- 在消息中 **原位显示生成按钮**，点击即可出图
- 如果开启了 **自动生成**，则直接后台出图、图文并茂

除此之外，你还可以：
- 点击 **悬浮按钮** → 输入 prompt 快速生成
- 对任意已有消息重新触发生图

## 🌟 核心能力

### 行内无感体验
- **消息内嵌渲染**：生成按钮和图片直接出现在对话消息流中，不弹窗、不遮挡，聊天和出图两不误
- **实时状态反馈**：请求排队 → 生成中 → 下载完成，全程可视化进度播报
- **沉浸式画廊**：点击内嵌缩略图打开全屏查看器，支持滚轮缩放、鼠标拖拽、左右翻页浏览历史记录
- **图片持久缓存**：所有生成图片通过 IndexedDB 本地缓存，刷新浏览器后仍可找回，支持设置自动过期天数

### NovelAI 深度集成
专为 NAI V4.5 系列模型深度适配：
- **完整高级参数**：支持 `Variety+`、`SMEA`、`SMEA DYN` 等 NAI 专属进阶开关
- **Vibe Transfer**：多图氛围特征提取 + 单图精确参考 (Precise Reference)，可导出/导入 `.naiv4vibe` 特征包
- **Encode-Vibe 能力探测**：自动检测当前后端是否支持 `/ai/encode-vibe` 接口
- **三种连接方式**：官方直连、RBQ 接口（支持免费层级自动限制）、自定义端点
- **高级解压引擎**：内置独立的图片解压机制，绕过网络传输导致的黑图 / 碎图问题

### ComfyUI 自定义工作流
- **工作流模板系统**：支持导入完整的 ComfyUI API 格式 JSON 工作流，可保存多套方案并一键切换
- **魔法变量注入**：在工作流 JSON 中使用 `{{prompt}}`、`{{negative_prompt}}`、`{{model}}`、`{{width}}`、`{{height}}`、`{{steps}}`、`{{cfg}}`、`{{sampler}}`、`{{scheduler}}`、`{{seed}}` 等占位符，每次生图自动填充
- **自动获取后端参数**：一键拉取 ComfyUI 可用的 Checkpoint 模型列表、采样器与调度器

### 插件生态
- **内置插件市场**：支持第三方功能插件的浏览、安装、更新与卸载
- **自定义模式注册**：插件可通过 `window.RBQ.api.registerMode()` 注册全新的生图后端（如 SD-WebUI、自建 API 等），注册后自动出现在模式下拉菜单中，共享通用参数面板
- **插件 API**：通过 `window.RBQ` 暴露 Hook、面板注册和设置读写接口，第三方开发者可深度扩展功能
